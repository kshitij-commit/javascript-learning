String and Template Literals

In JavaScript, strings are used to represent text. They can be created using single quotes (' '), double quotes (" "), or backticks (` `). Template literals, introduced in ES6, are a special type of string that allows for embedded expressions and multi-line strings.

  • Creating Strings:
    • Single Quotes: 'Hello, World!'
    • Double Quotes: "Hello, World!"
    • Template Literals: `Hello, World!`
  • Template Literals Features:
    • Embedded Expressions: You can embed expressions inside template literals using the syntax ${expression}. For example: 
      const name = 'John';
